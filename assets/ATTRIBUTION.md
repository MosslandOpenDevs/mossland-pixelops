# Asset Attribution

All assets used in Mossland PixelOps must be listed here with their license information.

## Format

```
### Asset Name
- **Source:** [URL]
- **Author:** [name]
- **License:** [license name + URL]
- **Modifications:** [none / description of changes]
```
